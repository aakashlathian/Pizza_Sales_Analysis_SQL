# Pizza_Sales_Analysis_SQL
This project analyzes a complete pizza store sales dataset using SQL to uncover insights about customer ordering behavior, revenue performance, and product trends. The dataset contains 48,620 order line items, 21,350 customer orders, 32 unique pizza types, and 4 pizza categories with detailed pricing and ingredient information.

The dataset contains:

21,350 orders

48,620 order line items

32 pizza types

4 pizza categories

Multiple pizza sizes and pricing tiers

The project demonstrates real-world data analytics skills using SQL: joins, aggregations, window functions, time-based analysis, ranking, and revenue insights.
